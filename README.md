## Installation

### 1. Clone the project
```bash
  git clone https://github.com/cdfornari/supermarket-reports.git
```

### 2. Go to the project directory
```bash
  cd supermarket-reports
```

### 3. Install dependencies to the server
```bash
  npm start
```
